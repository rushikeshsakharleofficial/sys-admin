How It Works

1

Point at a URL

Run /website-ui-deep-qa https://yoursite.com inside Claude Code. No config needed.

2

46 Checks Run

Playwright tests layout, accessibility, security, SEO, forms, and performance across 5 viewports. Artifacts saved to qa-artifacts/.

3

Get a Fix Plan

AI generates a prioritized fix-plan.md with every issue ranked by severity. Ship with confidence.
